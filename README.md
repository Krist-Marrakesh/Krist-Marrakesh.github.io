# aarkhipochkin.github.io

Личная страница на чистом HTML/CSS — без Jekyll, без сборки, без зависимостей.

## Файлы

- `index.html` — главная (био + кнопки)
- `projects.html` — список проектов
- `style.css` — общие стили (тема: Dracula, фиолетовый акцент)

## Как опубликовать через GitHub Pages

1. Создай репозиторий с именем **`<твой-логин>.github.io`** (например `Krist-Marrakesh.github.io`) — это обязательное имя для пользовательского сайта.
2. Залей в него три файла из этой папки (`index.html`, `projects.html`, `style.css`) в корень репозитория.
3. Зайди в **Settings → Pages**, в Source выбери ветку `main` и папку `/ (root)`, сохрани.
4. Через минуту-две сайт будет доступен по адресу `https://<твой-логин>.github.io`.

## Что можно подправить

- Аватар берётся напрямую с GitHub: `https://github.com/Krist-Marrakesh.png`. Если сменишь логин — поменяй ссылку в `index.html`.
- Чтобы добавить новый проект, скопируй один блок `<a class="project-row">…</a>` в `projects.html` и поменяй ссылку, заголовок, описание и теги.
- Цвета — в `style.css` в самом верху, в блоке `:root`.
