fractalov — Project-from-Samsung-Campus-Innovation ↗

An Android app for interactive fractal exploration (Mandelbrot, Julia, Burning Ship, Multibrot) with its own formula DSL for custom fractals, on-device rendering with progressive refinement, and a Spring Boot backend that acts as the math source of truth. Built for Samsung Campus Innovation.

Kotlin Jetpack Compose Java / Spring Boot PostgreSQL
Classification-of-astronomical-objects ↗

A multi-class classification pipeline that sorts celestial objects — stars, quasars, galaxies, white dwarfs, red giants, exoplanet candidates — using CatBoost tuned with Optuna, rare-class-aware K-Fold validation, temperature scaling, and test-time augmentation.

Python CatBoost Optuna scikit-learn
DESKAPP — Productivity System ↗

A personal productivity backend that pulls in tasks, events, and notes from email, Telegram, and direct input, then runs them through an LLM pipeline for entity extraction and date normalization. Includes auth, calendar sync, and a Telegram bot front end.

FastAPI PostgreSQL Celery LLM React
Hybrid-Hotel-Recommendation-System ↗

A production-style hotel recommender that blends a user's social signals with their own preferences. Candidates are pulled from friends' ratings, embedding similarity, and popularity fallback, then ranked by a custom Deep & Cross Network with residual blocks, served as a containerized FastAPI service.

PyTorch FastAPI Docker Recommender Systems
MLYANDEX — Deepfake Detection ↗

Research into telling real faces from GAN-generated ones, built around how StyleGAN's upsampling leaves behind high-frequency noise and checkerboard artifacts. A custom HRNet (no-pooling stem, parallel high/low-resolution branches) outperformed a plain ResNet and a dual-stream RGB+FFT model, with weighted sampling and per-epoch threshold search to handle class imbalance.

PyTorch HRNet Computer Vision Jupyter
